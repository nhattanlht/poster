# Poster Bảo vệ Đồ án Tốt nghiệp - LearningPath

Đây là source code giao diện HTML cho Poster bảo vệ đồ án tốt nghiệp với đề tài **"Hệ thống hỗ trợ thông tin học tập cho sinh viên sử dụng AI Agents"**.

## 🚀 Hướng dẫn sử dụng (Dành cho thành viên nhóm)

### 1. Cách xem Poster
- Rất đơn giản, bạn chỉ cần mở file `poster.html` bằng bất kỳ trình duyệt web nào (như Google Chrome, Safari, Edge,...).
- Khuyến nghị: Do poster được thiết kế theo tỷ lệ của khổ giấy A0, bạn có thể thu phóng (Zoom) trình duyệt để xem các chi tiết rõ ràng nhất.

### 2. Cách xuất Poster ra file PDF (Để đem đi in)
- Khi mở file `poster.html` trên trình duyệt, bạn sẽ thấy một nút **"Tải PDF"** (hoặc biểu tượng tải xuống) ở góc trên bên phải màn hình.
- **Hoặc cách thủ công:** Bạn có thể nhấn `Ctrl + P` (hoặc `Cmd + P` trên Mac), chọn khổ giấy **A0** trong cài đặt máy in, bỏ tích "Headers and footers" (Tiêu đề và chân trang) và chọn **Lưu dưới dạng PDF** (Save as PDF).

### 3. Cấu trúc thư mục hiện tại
- `poster.html`: File chính chứa toàn bộ giao diện HTML, các lớp CSS (TailwindCSS) và nội dung văn bản.
- `images/`: Chứa toàn bộ các hình ảnh minh họa (giao diện website, mobile, sơ đồ hệ thống, biểu đồ...).
- `Logo PNG chất lượng/`: Chứa ảnh Logo độ phân giải cao của Trường và Khoa.

### 4. Cách chỉnh sửa nội dung (Nếu muốn)
- Bạn có thể mở file `poster.html` bằng các trình soạn thảo code (Visual Studio Code, Cursor,...).
- Nội dung đã được chia thành các phần rõ ràng, nằm gọn trong các khối `<!-- Comment -->`. Ví dụ: `<!-- Thực trạng & Mục tiêu -->`, `<!-- Các hệ thống tương tự -->`. Bạn chỉ cần Ctrl+F tìm chữ và sửa trực tiếp.
